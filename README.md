# RatKingGaming Website

GitHub Pages ready static site.

## Upload to GitHub
1. Extract this ZIP.
2. Open the extracted folder.
3. Upload **everything inside this folder** to your GitHub repository.
4. In GitHub: Settings → Pages → Deploy from branch → main → root.

Open `index.html` and replace:

`https://www.paypal.com/paypalme/REPLACE-ME`

with your real PayPal link.

## Add videos
Find the `Rat Pack Clips` section in `index.html` and replace the placeholder video cards with YouTube/Twitch embeds or links.
